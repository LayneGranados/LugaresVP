# LugaresVP

LugaresVP es una aplicación para  de zonas comunes en lugares públicos o privados, el backend esta construido en java, tiene dos aplicaciones clientes una App nativa en Android y otra en AngularJS que funciona como administrativa

  - Recolecte información sobre superviciones de sitios desde cualquier dispositivo android
  - Administre toda la información desde una interfaz web
